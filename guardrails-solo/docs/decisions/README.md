# Architecture Decision Records

Short documents that record significant decisions made during development.

## When to create one

- Authentication and session strategy
- Encryption algorithm or library choice
- Any third-party service that will receive user data
- Data retention and deletion approach
- Deviation from any rule in CLAUDE.md

Run `/create-adr` in Claude Code — it will fill in the template automatically.
